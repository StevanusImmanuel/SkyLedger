# Tech Stack

- Next.js
- React
- TailwindCSS
- shadcn/ui
- Framer Motion
- Lucide React

# Preferred UI Libraries

- shadcn/ui → base components
- React Bits → animated UI sections
- Magic UI → premium interactive components
- 21st.dev → inspiration and reusable modern components
- Framer Motion → custom animations

# UI Style

- Modern SaaS aesthetic
- White mode first
- Glassmorphism
- Smooth animations
- Minimal but premium

# Color Pallete 
- Neutral White : #ffffff
- Neutral Black : #000000
- Background :#f8fafc
- Main : #1a2d5a
- Secondary : #2170E4


# Rules

- Prefer reusable components
- Use Tailwind only
- Avoid inline CSS
- Use shadcn components before custom components
- Use React Bits/Magic/21st UI for animations when possible
